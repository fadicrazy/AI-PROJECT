# Breast Cancer AI — Multimodal Prediction System

A production-ready Flask application for breast cancer risk prediction using tabular ML models and an image-based CNN. The system includes a polished dark theme UI, model comparison dashboards, demo sample predictions, and an AI assistant for clinical context.

## Features

- Uses the Wisconsin Breast Cancer dataset from `sklearn.datasets`
- Trains 5 tabular classification models:
  - Random Forest
  - Gradient Boosting
  - Support Vector Machine (SVM)
  - Logistic Regression
  - K-Nearest Neighbors (KNN)
- Includes a TensorFlow/Keras CNN for image-based prediction
- Provides 8+ REST API endpoints for health checks, predictions, analytics, and AI chat
- Fully responsive dark-themed UI with Chart.js visualizations
- Image upload accepts JPG, PNG, WEBP, BMP
- Rule-based AI chat for dataset, model, and recommendation guidance

## Setup

1. Create and activate a Python environment.
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Train the tabular models:
   ```bash
   python train_models.py
   ```
4. Train the CNN image model:
   ```bash
   python train_image_model.py
   ```
5. Run the app:
   ```bash
   python app.py
   ```

Or start everything with:

```bash
python run.py
```

## Endpoints

- `GET /api/health`
- `GET /api/feature_names`
- `GET /api/dataset_stats`
- `GET /api/model_results`
- `GET /api/compare_models`
- `GET /api/cnn_metrics`
- `POST /api/predict`
- `POST /api/predict_image`
- `GET /api/predict_sample`
- `POST /api/agent/chat`

## Notes

- All predictions are for research/demo use only.
- The AI agent and report generator are rule-based and not a substitute for clinical advice.
- The frontend is a single-file dark UI built without React.
